Description general:

The Tiger Puzzle game shall be developed to provide a graphical interface in which the user (the prisoner) is going to be able to play a game call “Tiger Puzzle”,  The user has to click in one of the doors. Behind of the two door can be either a tiger or a lover.

The goal the Tiger Puzzle is that the User shall be able to find a solution following the directions of the jailer, and taking in account the statements for each door and the jailer.
